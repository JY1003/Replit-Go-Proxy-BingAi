## 在Replit部署[Go-Proxy-BingAI](https://github.com/adams549659584/go-proxy-bingai)
### 项目简介：
基于微软 New Bing 用 Vue3 和 Go 简单定制的微软 New Bing 演示站点，拥有一致的 UI 体验，支持 ChatGPT 提示词，国内可用，基本兼容微软 Bing AI 所有功能，无需登录即可畅聊。
